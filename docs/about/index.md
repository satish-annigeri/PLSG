# About PLSG

The Python Learning Support Group (PLSG) is an initiative of the Structral Engineers' Associaion-India (SEA-India).

It is a voluntary group of members interested in learning Python programming for structural engineering applications. It is led by a senior structural engineer and the members currently primarily use Microsoft Excel but wish to explore using Python.

The SEA-India is a professional body registered under the [_The Karnataka Societies Registration Act, 1960_](https://dpal.karnataka.gov.in/storage/pdf-files/acts%20alpha%20and%20dept%20wise%20acts/17%20of%201960%20(E).pdf). Membership to the SEA-India is open to anyone with a postgraduate degree in structural engineering or allied fields.
