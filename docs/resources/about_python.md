# About Python

Python is a high-level, general purpose programming language. It has the following features:

1. It is an interpreted language
2. It is dynamically type-checked and garbage-collected
3. It supports multiple paradigms,including procedural, object oriented and functional
4. Python emphasizes code readibility with the use of significant indentation (indentation is required, not optional)
5. It is an interpreted programming language
6. Its current version is 3, with the latest release being 3.14.3 (in March 2026)
7. It has a rich set of built-in data types such as list, tuple, set, dictionary
8. In Python, everything is an object, as a result not only variables but functions are also objects
9. When using object oriented programming in Python, there are a large number of special built-in methods, called dunder methods that help in overloading operators
10. Python is known for the large number of built-in and thrid-party packages that make even complex tasks easy
